> * 举几个训读词的例子：  
>   * 八百万（やおよろず）：千千万万，无数
>   * 千歳（ちとせ）：千年。很长的年月
>   * 百千鳥（ももちどり）：非常多，各种各样的鸟
>   * 九重（ここのえ）：九层。比喻皇宫
>   * 二目（ふため）：再看，第二次看
> * 电话号码的读法
>   * 在读电话号码时，一般用上面列出的常用发音
>   * 4不读为し
>   * 9不读为く
>   * 0读为ゼロ或者まる而不读成れい
>   * “‐”读の
> * 7的读法
>   * しち、いち发音比较像，所以7一般用なな，避免辨别困难
> * 少用读音的使用
>   * よん、なな、きゅう　是一组（常用组）
>   * し　、しち、く　　　是一组（少用组）
>   * 对某种事物进行描述时，一般读法会保持一致用同一组的读法，
>       比如读月份的时候：しがつ、しちがつ、くがつ，就用的少用读音这一组


## 组合数字的读法
### 大数的读法
日语中数字的读法与中文一样。按万位分组，每组分个，十，百，千四位。读一个数加一个单位，读一组数加一个单位。

### 小数和负数
**小数**  
先读出小数点前面的数，再读小数点（てん），再依次读出小数点后面的数字
> 0.0059　ゼロてんゼロゼロごきゅう  
> 3.14　　さんてんいちよん

**负数**  
读数字之前先读负号（マイナス）  
> -18　マイナスじゅうはち

> ```
> * 为什么日语中的日期是这样的呢？
>     古代表示天数的「日」读作「か」，日本固有的数字说法（1~10：ひ、ふ、み、よ、いつ、む、なな、や、この、とお）＋「か」就构成了日期的说法。
> * 1~10的日期来源于和语，那十日以后呢？
>     和语十日以后的日期是用加法来表示的。
>     比如十一日，古代说成「とをかあまりひとひ」也就是“十日余一日”，那么以此类推，十二日、十三日就要分别说成「とをかあまりふつか」「とをかあまりみか」。
>     再比如二十日说成「はつか」，那么二十一日自然就说成「はつかあまりひとひ」“二十日余一日”。
> * 那三十日呢？三十日有训读吗？
>     三十日也有，是「みそか」
>     在古代数字中，「そ」是“十”的意思，比如三十、四十、五十，古文中分别读作「みそ」「よそ」「いそ」，那么「みそ+か」，就是三十日的读音了。（十：とを　二十：はた）
> * 和语的计数也太麻烦了吧？
>     对，用日本原有的计数的话，会很啰嗦，所以在汉语传入日本后，将十日以后的读法改为了汉语的「じゅういちにち」「じゅうににち」了。
> ```

# 计量
## 音便
### 数字的变化
1. 数字是2，5，7，9时，永不音便
2. 数字是1，6，8，10时，如果发生音便，那么数字会变成促音 

| 数字  | 变形           |   k   |   s   |   t   |   h   |
| :---: | :------------- | :---: | :---: | :---: | :---: |
|   1   | いち　→ いっ   |  か   |  さ   |  た   |  は   |
|   6   | ろく　→ ろっ   |  か   |       |       |  は   |
|   8   | はち　→ はっ   |  か   |  さ   |  た   |  は   |
|  10   | じゅう→ じゅっ |  か   |  さ   |  た   |  は   |
```
当数字1，6，8，10遇到量词的开头是k,s,t,h行的假名时，会发生音便。
6有点特殊，在遇到s,t行开头的假名时不音便。
は行的は、ひ、ふ、へ、ほ会变成ぱ、ぴ、ぷ、ぺ、ぽ。
举几个栗子：
六回　ろっかい  要音便，因为【ろく＋回】中的回是か开头
六枚　ろくまい  不音便，因为【ろく＋か・は行音开头的量词】才音便，枚不是か・は行开头的量词
六歳　ろくさい  不音便，同上
八歳　はっさい  要音便，因为【はち＋歳】中的歳是さ开头
一杯　いっぱい　要音便
六匹　ろっぴき　要音便
```
3. 数字是4时，如果发生音便，一定是将「よん」变成「よ」。量词几乎不发生变化，只有很少一些例外
```
四時　よじ
四年　よねん
四畳　よじょう
```
4. 数字是3时，如果发生音便，数字不变，量词变为浊音
```
三百　さんびゃく　　何百　なんびゃく
三千　さんぜん　　　何千　なんぜん
三本　さんぼん　　　何本　なんぼん
```
5. 当数字是「何」并且读成「なん」时，音便的规则和数字3一样（都是「ん」结尾） 
### 量词的变化
* 数字1（いち）、6（ろく）、8（はち）、10（じゅう）后面接以「か、さ、た、は」行假名开头的量词，会发生促音便。注意：「か、は」行假名时1、6、8、10均会发生促音化，「さ、た」行假名时只有1、8、10会发生促音化，数字6不发生变化。且接「は」行假名会发生半浊音便。
	```
	回（かい）：いっかい、ろっかい、はっかい、じゅっかい
	個（こ  ）：いっこ　、ろっこ　、はっこ　、じゅっこ
	歳（さい）：いっさい、はっさい、じゅっさい
	頭（とう）：いっとう、はっとう、じゅっとう
	杯（はい）：いっぱい、ろっぱい、はっぱい、じゅっぱい
	匹（ひき）：いっぴき、ろっぴき、はっぴき、じゅっぴき
	本（ほん）：いっぽん、ろっぽん、はっぽん、じゅっぽん
	```
* 数字3（さん）后接「は」行假名的量词时，量词的第一个音变为浊音。
	```
	杯（はい）さんばい 　
	匹（ひき）さんびき
	```
### 拓展
其他一般音便规则：
* 以「く、き」为尾音的汉字与第一个音为「か」行假名的汉字结合成新词时，一般会发生促音便。
	```
	学「がく」＋校「こう」＝学校「がっこう」
	石「せき」＋鹸「けん」＝石鹸「せっけん」
	```
* 以「つ、ち」为尾音的汉字与第一个音为「か、さ、た、は」行假名的汉字结合成新词时，一般会发生促音便，且「は」行假名会同时发生半浊音便。
	```
	発「はつ」＋達「たつ」＝発達「はったつ」
	雑「ざつ」＋誌「し」　＝雑誌「ざっし」
	一「いち」＋緒「しょ」＝一緒「いっしょ」
	失「しつ」＋敗「はい」＝失敗「しっぱい」
	```
* 前一汉字以ん结尾，后一汉字首音为「か、さ、た、は」行时会发生浊音便。且以「は」行开头的多变为相应的半浊音。
	```
	満「まん」＋足「そく」＝満足「まんぞく」
	番「ばん」＋組「くみ」＝番組「ばんぐみ」
	近「きん」＋所「しょ」＝近所「きんじょ」
	遠「えん」＋方「ほう」＝遠方「えんぽう」
	先「せん」＋輩「はい」＝先輩「せんぱい」

	注意：上述规律并非绝对，不发生浊音化的词也不少。
	勉「べん」＋強「きょう」＝勉強「べんきょう」
	間「かん」＋単「たん」  ＝簡単「かんたん」
	洗「せん」＋濯「たく」  ＝洗濯「せんたく」
	```
* 在复合词中，后续词的首音在「か、さ、た、は」行上时，一般由清音变浊音。
	```
	灰「はい」＋皿「さら」　＝灰皿「はいざら」
	台「だい」＋所「ところ」＝台所「だいどころ」
	草「くさ」＋花「はな」　＝草花「くさばな」
	一人「ひとり」＋部屋「へや」　　　＝一人部屋「ひとりべや」
	株式「かぶしき」＋会社「かいしゃ」＝株式会社「かぶしきがいしゃ」
	```
* ABAB式复合词中，因前后词重叠紧密，多发生浊音便。
	```
	時「とき」＋時「とき」＝時々「ときどき」
	人「ひと」＋人「ひと」＝人々「ひとびと」
	様「さま」＋様「さま」＝様々「さまざま」
	```
## 量词
